Jan 2023 - April 2023

Book Recommender System using Artificial Neural Network and Embedding

Five diverse Gutenberg digital books were selected from different authors within the same genre, ensuring semantic similarity. The data underwent preprocessing to remove stop words and irrelevant terms. Subsequently, 200 random document samples were created for each book, with 100-word records labeled accordingly. Feature engineering techniques such as Bag of Words (BOW), term frequency-inverse document frequency (TF-IDF), and N-gram were employed. The data was then trained using support vector machines (SVM), decision trees, k-nearest neighbors, and XG-Boost to enable the machine to recognize book authors and genres. Model performance was evaluated through ten-fold cross-validation. Error analysis was conducted to identify characteristics of instance records causing misclassification and to examine potential biases related to authors or books.The models were optimized using TensorFlow, Keras, and sci-kit-learn. An Artificial Neural Network (ANN) model with embedded layer values was trained to characterize books effectively. Cosine similarity was employed to identify books with similar characteristics, boosting recommendation accuracy. By combining clustering techniques and visualizing results, the system ensures tailored book recommendations based on individual user interests, thus enhancing the overall user experience

September 2022 - December 2022

Data analysis using R

In this project, I addressed customer churn prediction for a telecoms provider, a critical issue given the high churn rates in some markets. Firstly, I partitioned the provided customer churn dataset using the holdout method, allocating 67% of the records to the training dataset and 33% to the test dataset, visually confirming the proportions with a bar graph. Next, I rebalanced the training dataset to ensure 20% of the records had true churn values, calculating the necessary number of true churn records to resample accordingly. After rebalancing, I confirmed the proportion of true churn records in the dataset. Moving forward, I developed a decision tree model to predict churn using appropriate predictors from the dataset. Subsequently, I employed an ensemble method, such as Random Forest or Adaboost, tuning hyperparameters to enhance performance and compared it against the decision tree model. Utilizing a confusion matrix, I evaluated the models based on accuracy, sensitivity, and specificity, identifying the best and worst performers for each criterion. Finally, I conducted a ROC analysis to compare the performance of the ensemble method with the decision tree technique, plotting the ROC graph to visualize and interpret model performance. This project showcases my proficiency in predictive modeling techniques and evaluation methodologies, crucial skills for data-driven decision-making in telecoms and other industries.
